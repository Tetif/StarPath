# 3D models

## `starship.glb`

Low-poly SpaceX Starship–style rocket mesh for flight visualization.

- **License:** CC0 (public domain) — generated for this project via `scripts/generate_starship_glb.py`
- **Scale:** ~50 m tall (real Starship dimensions)
- **Axes:** origin at base; **+Y** is the flight direction (nose up)
- **Regenerate:** `python scripts/generate_starship_glb.py`
